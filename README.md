# Mermaid AI editor

Create and edit Mermaid flow charts with AI (Claude).

You can upload text, code files, PDF and Word documents.

---

**How to use**

Add your `ANTHROPIC_API_KEY` to the `.env` file in the server directory and you're good to go!
